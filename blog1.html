What is Pseudo Classes ?

A pseudo-class is used to define a special state of an element.

pseudo-class, it can be attached to any selector with a colon and accepts classes, IDs, and HTML tags as parameters.

* Style an element when a user mouses over it

* Style visited and unvisited links differently

* Style an element when it gets focus

There are different pseudo-classes in CSS..

Dynamic pseudo-classes. :link. :visited. :hover. :active. etc

The target pseudo-classes. :target.

The language pseudo-classes. :lang.

The UI element states pseudo-classes. :enabled. :disabled. :checked.

Structural pseudo-classes. :root. :nth-child. :nth-last-child. etc

The negation pseudo-classes. :not.

1) : Has ()

The class .selector only gets selected if it contains the elements passed to it as parameters using the has() pseudo-class:

Example

.selector:has(.class) { ... }

.selector:has(#id) { ... }

.selector:has(div) { ... }

2) : nth-child()

The :nth-child() CSS pseudo-class matches elements based on their position among a group of siblings.

Example

li:nth-child(-n + 3) { border: 2px solid orange; margin-bottom: 1px; }

li:nth-child(even) { background-color: lightyellow; }

3) :nth-of-type()

The :nth-of-type() CSS pseudo-class matches elements based on their position among siblings of the same type (tag name).

Example

.selector :nth-of-type(even) { border: 2px solid orange; }

4) :placeholder-shown

The :placeholder-shown CSS pseudo-class represents any input or textarea element that is currently displaying placeholder text.

Example

input:placeholder-shown { background-color: ivory; border: 2px solid darkorange; border-radius: 5px; }

5) :where()

The :where() CSS pseudo-class function takes a selector list as its argument, and selects any element that can be selected by one of the selectors in that list.

Example

:where(ol, ul) :where(ol, ul) ol { list-style-type: lower-greek; color: chocolate; }

6) :not()

The :not() CSS pseudo-class represents elements that do not match a list of selectors. Since it prevents specific items from being selected, it is known as the negation pseudo-class.

Example

p > :not(strong, b.important) { color: darkmagenta; }
